# Pathlight Community Services GitHub
Pathlight Community Services is a 401(c)3 non-profit that provides Adult Foster Care services to hundreds of clients across five counties in Southeast Michigan. 

## About This Page
The public repos on this organization's GitHub are the primary static sites used by the public.
This currently includes:
- mipath.org: The main site for information about Pathlight Community Services.
- htf.mipath.org: The site for the Hoarding Task Force of Washtenaw, a hoarding and clutter cleaning volunteer service provide by Pathlight.

The private repos are current internal system apps that are in constant development.
